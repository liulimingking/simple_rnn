# 核心步骤：  
加载数据：载入 IMDB 数据集。  
数据预处理：由于影评长度不一，需要将所有序列填充或截断到相同的长度。  
构建模型：定义一个简单的 RNN 模型架构，通常包括嵌入层 (Embedding)、RNN 层 (SimpleRNN) 和一个用于分类的全连接层 (Dense)。  
编译模型：为模型指定优化器、损失函数和评估指标。  
训练模型：使用训练数据来训练模型。  
评估模型：使用测试数据来检验模型的准确率。  


# 代码解释  
嵌入层 (Embedding): 这是处理文本数据的第一步。它将每个单词的整数索引映射到一个固定维度的“词向量”上。这个向量可以捕捉单词的语义信息，相似的单词会有相似的向量。  
SimpleRNN 层: 这是 RNN 的核心。它会依次处理序列中的每一个元素（在这里是每个单词的词向量），并维护一个“隐藏状态”。这个隐藏状态会从序列的开头传递到结尾，从而捕捉整个序列的信息。  
全连接层 (Dense): 在 RNN 处理完整个序列后，我们得到一个最终的输出向量。这个向量被送入一个全连接层，进行最后的分类。  
Sigmoid 激活函数: 由于这是一个二分类问题（正面/负面），输出层使用 sigmoid 函数非常合适。它会输出一个 0 到 1 之间的值，我们可以将其解释为“正面评论”的概率。  
序列填充 (pad_sequences): 这是 RNN 数据预处理中非常关键的一步。因为神经网络的输入需要是形状统一的张量，所以我们需要将长短不一的评论都处理成相同的长度。
